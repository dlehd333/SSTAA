# SSTAA : Subway Station Trading Area Analysis
# 개요

- C# .NET Framework와 EntityFramework 및 DevExpress를 이용해 서울시 내 각 구별 상권분석용 프로그램을 개발하였다.

# 개발 기간

- 2020년 12월 28일 ~ 2021년 01월 08일

# 참여 인원

- IoT기반 스마트팩토리 SW개발 전문가과정
- 김대근, 안성윤, 이동희 총 3명
